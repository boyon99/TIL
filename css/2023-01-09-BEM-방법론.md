## BEM (block element modifier) 방법론

 #### `__` 기호로 요소의 일부분을 표시한다. 
 ```html
  <div class="container">
    <div class="container__name"></div>
    <div class="container__value"></div>
  </div>
```

#### `--` 기호로 요소의 상태를 표시한다.
```html
  <div class="btn--success"></div>
  <div class="btn--error"></div>
  <div class="brn--fail"></div>
```
