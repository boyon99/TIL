## 리터럴 (literal)
사람이 이해할 수 있는 문자 또는 약속된 기호를 사용해 값을 생성하는 표기법(notation)

| 리터럴 | 예시 |
| --- | --- |
| 정수 리터럴 | `100` |
| 부동소수점 리터럴 | `10.5` |
| 2진수 리터럴 | `0b0100001` |
| 8진수 리터럴 | `0o101` |
| 16진수 리터럴 | `0x41` |
| 문자열 리터럴 | `“hello”` `'world'` |
| 불리언 리터럴 | `true` `false` |
| null 리터럴 | `null` |
| undefined 리터럴 | `undefined` |
| 객체 리터럴 | `{name : ”Lee”}` |
| 배열 리터럴 | [`1, 2, 3]` |
| 함수 리터럴 | `function() {}` |
