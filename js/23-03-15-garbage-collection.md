## 가비지 컬렉션 Garbage Collection

자바스크립트의 메모리 관리 방법으로, 자바스크립트 엔진이 자동으로 데이터가 할당된 메모리에서 더 이상 사용되지 않는 데이터를 해제하는 것을 의미한다. 가비지 컬렉션은 자동으로 동작하기 때문에, 개발자가 직접 강제 실행하거나 관리할 수 없다.

```js
let a = { x: 1 };
let b = a;

b.x = 2;
console.log(b); // { x: 2 }
console.log(a); // { x: 2 }

// 가비지 컬렉션을 통해 숫자 1은 메모리에서 해제
```
